## 计算需要缴纳的个税
根据实际情况修改以下变量：
```
var (
	a      float64 = 18000 // 养老保险缴纳的基数
	b      float64 = 18000 // 医疗保险缴纳的基数
	c      float64 = 18000 // 公积金缴纳的基数
	salary float64 = 20000 // 每个月的实际工资
)
```
然后在main方法调用，如下示例：
```go
func main() {
	// 计算需要缴纳的个税和公司缴纳的公积金
	f1, f2 := test1(a, b, c, salary)
	fmt.Printf("需要缴纳的个税：%v￥，公司缴纳的公积金：%v￥\n", f1, f2)
}
```
